# **App Name**: GDGExecutor

## Core Features:

- Code Input: Users can upload, write, or paste code snippets into the editor.
- Syntax Highlighting: Code snippets are displayed in an editor that offers syntax highlighting for major languages.
- Code Analysis Tool: Powered by a gen-AI, automatically detects the language of the inputted code and can explain it using reasoning. Also gives complexity and security ratings. These AI-driven code analyses are available at the user's request.
- Copy to Clipboard: Users can copy the results of the AI's code analysis with a single click for easy sharing and documentation.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to create a calm and intellectual environment.
- Background color: Very light lavender (#F5F5FF) provides a subtle and non-distracting backdrop.
- Accent color: Pale blue (#ADD8E6) complements the primary color and provides contrast for key actions.
- Body and headline font: 'Inter' sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text.
- Code font: 'Source Code Pro' monospaced font for displaying code snippets clearly.
- Simple, geometric icons that relate to code execution and analysis.
- Subtle animations during code processing and analysis to indicate activity.
